# Revanate

## Project Description
Revanate, a Java based ORM library simplifying connection to the PostgreSQL without the need for SQL or connection managment.

## Technologies Used

* PostgreSQL - version 42.3.3
* Java - version 8.0
* Apache commons - version 2.9.0
* JUnit - version 4.13.2
